Exercise 3
Task: Build a quiz feature based on an existing html page
Description

There is a website with XML multiple choice test questions as follows: Page

Description of the webpage:

  • The webpage displays a list of multiple-choice questions about XML.
  • Each question has 3 to 4 answer options, with the correct answer bolded.

Please analyze the webpage and build a quiz feature based on this webpage (using the knowledge learned to interact with the DOM).

Necessary features include:

  • Number the questions (starting from 1).
  • Remove the bold formatting from the correct answers.
  • Create a score table and display the questions. The score table should show: number of correct answers/total questions attempted. (To be displayed as shown in the image below).
  • When clicking on each question in the question table, the screen will scroll to the position of that question.
  • When answering a question (clicking on one of the answer options), that answer will be bolded, and the question table will display the result (green for correct answers, red for incorrect ones), calculate the score, and display it in the score table.

Please do not modify the file src\exercises\exercise_3\task.html.

Please write your answer in the file src\exercises\exercise_3\task.js.
